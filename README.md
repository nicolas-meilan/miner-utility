# Miner Utility ESP32

- Show estimated daily profit on ETH, USD and ARS.
- Show monthly profit on ETH, USD and ARS.

## LEDS
  - Green &rarr; P12 220 OHM - GND
  - RED &rarr; P13 220 OHM - GND

## Button
  - Pull Up 220 OHM &rarr; P32 - GND 5V

## LCD I2C
  - SDA &rarr; P23
  - SCL &rarr; P22
  - 5V - GND
